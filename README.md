SentiSteam is a Windows installer script for LuaTools that provides a menu to install, uninstall, display help, or exit.

Installation
Download or clone the repository. Place main.bat and install-plugin.ps1 in the same folder. Run main.bat to open the installer menu.

Usage
Run main.bat. Select option 1 to install LuaTools. Select option 2 to uninstall LuaTools. Select option 3 to view help and instructions. Select option 4 to exit the installer. Follow the on screen help instructions to complete the setup in Steam.

Help and Instructions
After installing LuaTools, open Steam. Log in if required. Open the plugin manager. Disable all plugins except LuaTools. Enable LuaTools and save changes. Restart Steam if necessary. Open a game page in Steam. Use the Add via LuaTools option to add games. Restart Steam after installation if needed.

Files
main.bat contains the installer logic. install-plugin.ps1 installs necessary LuaTools components.
